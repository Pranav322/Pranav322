#### [RAG Chatbot Application](https://github.com/Pranav322/rag-chatbot)
*A production-ready Retrieval Augmented Generation system for document interaction.*
- **Architecture:** Built a Django-based RAG pipeline using `pgvector` for semantic search and OpenAI for context-aware responses.
- **Technical Challenge:** Implemented an intelligent OCR & Vision pipeline (Tesseract) to extract and index text from multi-format documents (PDFs, DOCX, Images).
- **Stack:** Django, React, TypeScript, PostgreSQL (pgvector), Celery, Docker.

#### [YouTube Technical Note-Taker](https://github.com/Pranav322/youtube-notes)
*AI-powered pipeline to convert technical video content into structured documentation.*
- **Architecture:** Engineered a Map-Reduce AI processing flow to handle long-form video context without losing fidelity.
- **Features:** Includes syntax-highlighted code block extraction, PDF export, and custom rate-limiting middleware.
- **Stack:** FastAPI (Python 3.13), Next.js 16, Azure OpenAI, PostgreSQL, Docker Compose.

#### [Interactive Desktop Portfolio](https://github.com/Pranav322/portfolio-v3)
*A complex frontend engineering showcase blending GUI and CLI paradigms.*
- **Engineering:** Developed a custom window management system with drag-and-drop capabilities and state persistence using React context.
- **UX:** Seamlessly integrated a functional terminal interface with a desktop GUI environment using Framer Motion for hardware-accelerated animations.
- **Stack:** Next.js, TypeScript, Framer Motion, Tailwind CSS.
